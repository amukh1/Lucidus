# Installation

<br>

## To install and set up the library, run:

From Github:

```sh
$ git clone https://github.com/amukh1/Lucidus ./build
$ cd build
$ chmod +x ./install.sh
$ ./install.sh
$ chmod +x ./compile.sh ../run.sh
$ cp ./luci /usr/bin
```

From APT:
    
```sh
$ sudo apt install luci
```
